# SPECIFICATION

## Judul
Sistem Penyortiran Barang di Gudang Menggunakan Heap dan Hash Table

## Tujuan
Membuat sistem manajemen gudang berbasis terminal menggunakan C++ yang mampu:
- Menyimpan data barang.
- Mencari barang dengan cepat.
- Mengelola prioritas barang.
- Mengurangi keterlambatan pengiriman.
- Mengurangi risiko barang kedaluwarsa.

## Teknologi
- Bahasa: C++17
- UI: Terminal / CLI
- Struktur Data:
  - Hash Table
  - Min Heap

## Data Barang
- ID Barang
- Nama Barang
- Stok
- Lokasi Rak
- Prioritas

## Fitur
1. Tambah Barang
2. Cari Barang
3. Tampilkan Semua Barang
4. Tampilkan Barang Prioritas Tertinggi
5. Keluarkan Barang
6. Tampilkan Antrian Prioritas
7. Keluar

## Acceptance Criteria
- Barang dapat ditambahkan.
- Barang dapat dicari berdasarkan ID.
- Prioritas barang dikelola menggunakan Heap.
- Data barang disimpan menggunakan Hash Table.
- Seluruh sistem berjalan pada terminal.
